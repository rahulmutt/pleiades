> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Move the packaged-ephemeris range *claim* from 1500–2500 CE to 1600–2600 CE everywhere it is asserted in prose and in code-level labels (and the tests that pin them), retiring the obsolete 1500–1550 known-gap narrative.

**Architecture:** Pure wording/label change in two layers — Layer 1 is Markdown prose across the spec/plan/docs tree; Layer 2 is a handful of Rust display/label strings in `pleiades-data` and `pleiades-validate` plus the assertions that guard them in `pleiades-validate` and `pleiades-cli`. No numeric data, epoch constants, fixtures, behavior, or accuracy thresholds change. The boundary reference fixtures (a real 2600 CE ceiling) are deferred Layer 3, out of scope here.

**Tech Stack:** Rust (cargo workspace), Markdown. Verified with `cargo build`, `cargo test`, `cargo fmt --check`, `cargo clippy`.
