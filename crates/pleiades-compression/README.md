# pleiades-compression

[![crates.io](https://img.shields.io/crates/v/pleiades-compression.svg)](https://crates.io/crates/pleiades-compression)
[![docs.rs](https://img.shields.io/docsrs/pleiades-compression)](https://docs.rs/pleiades-compression)
[![License: MIT OR Apache-2.0](https://img.shields.io/badge/license-MIT%20OR%20Apache--2.0-blue.svg)](#license)

Compressed ephemeris artifact data structures and codec helpers for the [pleiades](https://github.com/rahulmutt/pleiades) astrology workspace.

Depends only on `pleiades-types`. Enable the `serde` feature for serialization support.

## Status

Experimental `0.2.x`. The packaged-data artifact built on these structures is draft-grade; see the [workspace README](https://github.com/rahulmutt/pleiades#readme) for the full maturity posture.
