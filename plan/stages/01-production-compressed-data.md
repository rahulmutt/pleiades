# Phase 1 — Artifact Accuracy and Packaged-Data Production

## Goal

Ship a production-quality compressed ephemeris artifact for 1500-2500 CE that satisfies `spec/data-compression.md`, `requirements.md` FR-9, and the packaged-backend responsibilities in `spec/backends.md`.

## Implementation goals

- Rework fitting and reconstruction until body/channel errors meet published thresholds for the advertised scope.
- Use validated public inputs from Phase 2 rather than ad hoc sparse samples for production fitting.
- Keep the finalized body-class/channel thresholds synchronized with source-fit and independent hold-out validation.
- Decide whether polynomial order, Chebyshev fitting, residual tables, per-body cadence, or channel-specific storage changes are required.
- Residual correction now uses channel-specific sample fractions so angular channels can keep the dense lattice while distance channels stay on the sparse lattice for most bodies, while selected asteroids and custom bodies now also use the dense distance residual lattice; the residual-channel lattice helper now explicitly centralizes the dense-body angular-channel rule while the regression coverage now includes selected asteroids and custom bodies, and the residual-sample regression now explicitly exercises both Ceres and a non-asteroid custom body. The dense distance residual lattice now also extends to inner and outer planets, so the major-planet distance channel is sampled with the same tighter residual cadence as the other dense distance bodies. The validation and outlier cadence now also treats inner and outer planets as dense, so the major-planet fit-drift reports sample those bodies more thoroughly without changing the candidate fit ladder. Completed: the outer-planet regression now explicitly covers Jupiter, Saturn, Uranus, and Neptune, pinning the shared medium fit ladder and the dense distance residual cadence across the full outer-planet class.
- Residual correction search now also explores all remaining channel-order permutations when composing channels, so order-sensitive candidates no longer depend on the previous suffix-only traversal, and equal-error candidates now prefer the smaller residual footprint. Equal-error candidate-versus-fallback decisions now prefer the simpler segment, and the candidate-vs-fallback path is now accuracy-first so a worse fit no longer wins purely because it is simpler.
- Higher-order fit candidates now share a control-point fallback helper across longitude, latitude, and distance channels, so fit reconstruction can degrade to a lower-order interpolation when quantization would otherwise overflow, and the fallback path now also tries a dense quarter-point control-point lattice before the exact-third fallback on irregular spans, with one-sixth/five-sixth probes now available on very long dense spans before the one-third/two-thirds fallback; when those sixth-point probes are unavailable, the split search still falls through to the one-third/two-thirds ladder instead of reverting directly to midpoint splitting. The most extreme dense-body spans now also consult an internal one-seventh/six-sevenths rung ahead of the one-fifth fallback, and the super-extreme dense-body ladder now also consults one-eighth/seven-eighths probes before that one-seventh rung, giving the split search one more dense option on spans far beyond the current corpus. The fallback reconstruction path now also tries a dense five-point polynomial lattice on the longest dense-body spans before the exact-third fallback when one-fifth through four-fifth samples are available. The release-facing generation-policy prose now also names the quarter-biased dense-body split posture alongside those probe ladders, and now also names lunar points in the dense-body candidate ladder so the long-form fit strategy stays aligned with the dense validation and residual-sampling rules. ## Completion criteria

Phase 1 is complete when a clean checkout can regenerate the packaged artifact and pass published reference/hold-out thresholds for the advertised bodies, channels, request modes, and 1500-2500 CE range.

## Out of scope

- Broadening body claims without Phase 2/3 evidence.
- Implementing apparent, topocentric, UTC/Delta-T, or native-sidereal behavior.
- Promoting catalog compatibility claims.
